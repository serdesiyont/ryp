# Rate Your Professors (RYP)

A free, open platform where students across Ethiopia share honest reviews of
universities and professors. Live at **[rypet.vercel.app](https://rypet.vercel.app)**.

Search any campus or lecturer, read real student experiences, and add your own
to help others make informed academic choices.

## Tech stack

Next.js (App Router, server components) · TypeScript · Tailwind CSS · MongoDB

## Getting started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:4000](http://localhost:4000).

## Contributing

Contributions are welcome — every bit makes the platform better for students.

- **Add a review** — rate a professor or university on the live site.
- **Report a bug** — open an [issue](https://github.com/serdesiyont/ryp/issues).
- **Contribute code** — fork, create a branch, and open a pull request:

  ```bash
  git checkout -b my-feature
  # make your changes
  git commit -m "describe your change"
  git push origin my-feature
  ```

  Then open a PR against `main`.
